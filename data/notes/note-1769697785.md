# Docker Basics

Docker lets you package an application with its dependencies into a single unit called an image.

## Core Concepts
- **Image**: Immutable blueprint
- **Container**: Running instance of an image
- **Layer**: Cached filesystem step

## Common Commands
```bash
docker build .
docker run -p 3000:3000 app
docker ps
